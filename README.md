# chat app
Chat app with react and socket.io.
[Check out the deployed site](https://chat-react-in.netlify.app/)

## Core Packages

1. react 
2. socket.io 
3. react-router 
4. express 

## Features

- Loging whit username
- multiple rooms
- multiple users

## How to setup locally

- Then run <code>npm install && npm start </code> .
